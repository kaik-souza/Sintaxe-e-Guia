	CSS Externo

<link rel="stylesheet" href=".css"> = para incorporar (stylesheet = folha de estilok)


	🎨 Seletores e Sintaxe Básica

selector { propriedade: valor; }
* = seletor universal
div, p = múltiplos seletores
#id = invoca por id
.classe = invoca por classe
elemento = invoca por tag
elemento1 elemento2 = descendente
elemento1 > elemento2 = filho direto
elemento1 + elemento2 = irmão adjacente
elemento1 ~ elemento2 = irmãos gerais
[attr] = atributo presente
[attr="valor"] = atributo com valor específico


	🖋️ Cores e Fundo

#ccc = abre o seletor
color: red; = cor do texto
background-color: blue; = cor de fundo
background-image: url('img.png'); = para inserir uma imagem de fundo
background-repeat: no-repeat; = para definir como a imagem de fundo será repitida ou não
background-size: cover | contain | auto; = como a imagem de fundo será redimensionada dentro do elemento
background-size: cover = a imagem cobre todo o elemento e mantém sua proporção
background-position: center | top | left | x y; = defini a posição da imagem
background-image: linear-gradient (direções, cores) = efeito de cor em gradiente em linha (degradê)
background-image: radial-gradient = efeito de cor em gradiente em circulo


	🔠 Texto e Fontes

@import url ('link')
cursor = altera o tipo de cursor
font-family: Arial, sans-serif; = tipo de fonte
font-size: 16px; = tam da fonte
font-weight: normal | bold | 100-900; = para determinar a grossura da fonte
font-style: normal | italic;
text-overflow = controla como o texto oculto é exibido quando ultrapassa o tamanho do elemento.
text-align: left | center | right | justify; = alinhamento do texto
text-decoration: none | underline | line-through; = para alterar a decoração do texto (comumente usado para links)
text-transform: uppercase | lowercase | capitalize; = determina o comportamento do texto (ex.: tudo maisculo ou minusculo)
text-indent: 50px = recua o inicio do parágrafo
text-shadow: 2px 2px 5px #red = sombra no texto
box-shadow = sombra nos elementos
line-height: 1.5; = espaçamento entre as linhas
letter-spacing: 2px; = espaçamento entre as letras
word-spacing: 4px; = espaçamento entre as palavras
direction: rtl = muda a direção do texto
unicode-bidi: bidi-override = deixa o texto ao contrário (usado mais para do mandarim ou arabe)


	📦 Box Model

width: 100px; height: 200px; = largura e altura
max-width, min-width, max-height, min-height; = maximo de largura e altura
padding: 10px; = espaçamento interno
margin: 10px; = espaçamento externo
margin-top = espaçamento parte de cima
border: 1px solid black; = borda (tamanho, tipo e cor)
border-left = esquerda da borda
border-bottom = parte baixa da borda
border-radius: 10px; = cantos arredondados
box-sizing: content-box | border-box;
box-sizing: border-box; = muda a forma como o tamanho de um elemento é calculado


	📐 Display e Layout

inline-block = fica na mesma linha que os outros elemento mas permite definir largura e altura
position: static | relative | absolute | fixed | sticky;
position: relative = torna possível passar parametros de posicionamento
position: fixed = deixa o posicionamento de um elemento fixo em determinado lugar da tela
position: absolute = posiciona um elemento em um local exato da tela ou do seu elemento pai
position: sticky = faz um elemento rolar normalmente na página e, ao atingir um ponto definido, ficar “grudado” na tela.
top, right, bottom, left = posicionamento
z-index: 10; = ordem de empilhamento (de -1 a 999)
overflow: visible = mostra o conteúdo mesmo que ultrapasse a área do elemento
overflow: hidden = mostra somente o que se encaixa dentro do elemento
overflow: scroll = acrescenta uma barra de rolagem para que seja possível olhar o conteúdo dentro do elemento
overflow: auto = acrescenta barra de rolagem se for necessário
float: left | right; clear: both = permite posicionar o elemento a esquerda e direita permitindo que ele flutue
visibility: visible | hidden;


	🧭 Flexbox (display: flex)

display inline = ocupa apenas o espaço do conteúdo em linha
display block = ocupa toda a largura disponível
display: flex = organiza os elementos de forma flexivel em linhas e colunas
display: block | inline | inline-block | flex | grid | none;
flex-direction: column = organiza os elementos em block 
flex-direction: row = organiza os elementos em inline
flex-direction: row || column-reverse = torna os elemetos em reverso
display inline = ocupa apenas o espaço do conteúdo em linha
display block = ocupa toda a largura disponível
display: flex = organiza os elementos de forma flexivel em linhas e colunas
display: block | inline | inline-block | flex | grid | none;
justify-content: flex-start | center | space-between | space-around; = alinhamento dos elementos na horizontal
justify-content: space-between = adapta os elementos dentro da div pai para que haja um espaçamento uniforme entre os elementos
justify-content: space-around = espaçamento uniforme em toda volta de cada elemento com responsividade   
align-items: flex-start | center | stretch | flex-end; = comportamento dos itens dentro de um container na forma vertical
flex-wrap: nowrap | wrap;
flex-wrap: wrap; = permite que os  intens quebrem a linha quando não couberem mais no container 
flex-flow: row wrap
align-content: center | space-between; comportamento dos elementos quando há quebra de linha
flex-grow: 1; = crescimento proporcional dos elementos para que se encaixe dentro do container
flex:15% = o item ocupará 15% do container
order: 1; = ordem dos itens
style="visibility" = define o estilo de visibilidade
style="clear:both" = limpa a flutuação
list-style-type = none; = nenhum estilo na lista
white-space = deixa o item que tem quebra de linha em uma linha só


	🔲 Grid Layout (display: grid)

display: grid;
grid-template-columns: 1fr 1fr 1fr;
grid-template-rows: auto 200px;
grid-column: 1 / 3; = ocupa colunas
grid-row: 2 / 4; = ocupa linhas
gap: 10px;
justify-items: start | center | end | stretch;
align-items: start | center | end | stretch;
place-items: center; /* alinhamento rápido */

	📱 Responsividade

  body { font-size: 14px; }
}
@media (orientation: landscape) { ... }
vw, vh, %, em, rem = unidades relativas
clamp(min, preferido, max) = valor fluido


	🖼️ Imagens e Objetos

object-fit: cover | contain;
object-position: center;

	🎯 Pseudo-classes

:hover = quando o mouse passa
:active = quando clicado
:focus = quando focado (inputs)
:first-child / :last-child
:nth-child(2) / :nth-child(odd/even)
:not(seletor) = nega seleção


	🎭 Pseudo-elementos

::before, ::after = adiciona conteúdo virtual
content: "texto" | "";
::first-letter = primeira letra
::first-line = primeira linha
::selection = parte selecionada pelo usuário

	🛡️ Boas Práticas

usar classes em vez de ids para estilos reaproveitáveis
organizar CSS em blocos lógicos
usar variáveis CSS (--cor-primaria: #333;)
preferir unidades relativas (em, rem, %) para responsividade
evitar !important sempre que possível


	📚 Recursos Rápidos

link externo: <link rel="stylesheet" href="style.css">
inline: <div style="color:red;">
import: @import url("outro.css");
variáveis: :root { --cor: blue; } -> color: var(--cor);
reset CSS (normalize.css) para consistência entre navegadores


	Sequência de Especificidade 

#id
tag.classe
classe
tag
!important = ignora as demais especificidades


	Midia Queries (responsividade)

/* Smartphones (600px para baixo) */
@media only screen and (max-width: 600px) {...}
/* Dispositivos um pouco maiores, mas ainda pequenos (Pequenos Tablets e Smartphones + largos, 600px para cima) */
@media only screen and (min-width: 600px) {...}
/*Dispostivos Médios (Tablets deitados, 768px para cima) */
@media only screen and (min-width: 768px) {...}
/* Dispostivos Largos (laptops/desktops, 992px para cima) */
@media only screen and (min-width: 992px) {...}
/* Super Largos (Telas maiores laptops, desktops e até TVs, 1200px para cima) */
@media only screen and (min-width: 1200px) {...}
